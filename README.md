# ⚖️ Compliance Contratual

> Monitoramento de compliance contratual com IA. Acompanha obrigações, prazos e renovações, alertando antes de vencimentos críticos.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/compliance-contratual/

## 📊 Resultados-chave (KPIs)
**monitor.** Obrigações · **0 perdidos** Prazos · **auto-alert** Renovações · **98%** Compliance

## 🧠 Tecnologias de IA
AI · NLP · Contract Lifecycle

## ✨ O que faz
- Rastreamento de obrigações
- Alertas de prazo e renovação
- Score de compliance
- Gestão de ciclo de vida

## 🗄️ Fonte de dados real
[Contratos corporativos](https://www.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Contratos corporativos
2. **Processamento com IA** — AI · NLP · Contract Lifecycle
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
